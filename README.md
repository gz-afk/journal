#  Daily Journal Web App

# ━━━━━━━━━━━━━━━━━━━━━━━

A simple and clean journaling web app built using HTML, CSS, and JavaScript.
Track your thoughts, mood, and daily spending — all in one place.

---

##  Features

*  Daily journal entries
*  Finance tracker (daily spending)
*  Emoji-based mood tracker
*  Interactive calendar (view past days)
*  Quote of the day
